# nwg-drawer

Es un menú de aplicaciónes creado por `nwg-shell`, [Web](https://github.com/nwg-piotr/nwg-drawer), muy completo, me ha gustado, ya que está optimizado para wayland.

# Instalación

Está en los repositorios oficiales, por lo que basta instalar desde pacman o paru:

```bash
sudo pacman -S nwg-drawer
```

# Configuración

La ruta de configuración se encuentra en: `~/.config/nwg-drawer/`

## Estilo o tema

El estilo o tema se encuentra en: `~/.config/nwg-drawer/drawer.css`, actualmente uso el tema por defecto tomado desde GitHub.

## Apertura de archivos

Con el click (izquierdo) se pueden abrir archivos con las aplicaciones definidas por defecto, para ello se usa el archivo `~/.config/nwg-drawer/preferred-apps.json`
