# WPAPERD

[wpaperd](https://github.com/danyspin97/wpaperd)
Es un daemon que se encarga de cambiar el fondo de pantalla, muy recomendado para Wayland y hyprland lo tiene dentro de su listado de utilidades.
Anteriormente he usado `swaybg` y `hyprpaper` pero ahora uso `wpaperd` ya que estaba buscando algo simple de configurar para cambio aletatorio, lo cual trabaja excelente.

## Instalación

En el caso de archlinux, basta con instalar el paquete `wpaperd` y ya está listo para usarlo.

```bash
sudo pacman -S wpaperd
```

## Configuración

El archivo de configuración se encuentra en la siguiente ubicación:

```bash
$HOME/.config/wpaperd/wpaperd.toml
```
